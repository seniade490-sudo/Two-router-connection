# Two-router-connection
Using Packet Tracer, I configured two routers, while doing so, I enabled password and password encryption, NO specific router configurations where made.
1. Connect R1 and R2 by thier GigabitEthernet0/0 interfaces;
2. Set the hostnames accordingly to the network diagram (R1 and R2).
3. Set the enable password on each router to 'cisco'
4. View the password in the running configuration, is it encrypted?
Enable Password encryption on each router
5. Enable password encryption on each router
6. View the password in the running configuration. Is it encrypted?
7. Disable password encryption on each router.
8. View the password in the running configuration. Is it encrypted?
